URLPathMapper

public class URLPathMapper<Route> where Route : RouteType

Router URL Path Mapper.

Note

All static paths are resolved before dynamic paths.

Usage:

// Static path
$0.map("/users") { .allUsers }

// Dynamic path
$0.map("/users/{id}/profile") { try .profile(withID: $0.param("id")) }
